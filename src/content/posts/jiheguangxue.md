# 几何光学基本原理

## 1. 概论 - 光学的发展

### 光学分类
- **几何光学**：基于光的直线传播研究光的传播及成像规律及其应用的科学。
- **物理光学**：以光是一种波动为基础的物理光学。
- **量子光学**：以光是一种粒子为基础的物理光学。量子光学一词是在有了激光后才提出来的。


## 1. 概论 - 光学发展历史

### 古代光学 (15世纪以前)
对简单光现象进行了记载并做了不系统的研究，制造了简单的光学仪器 (如平面镜、凸面镜、凹面镜)。e.g. 凹面镜取火。

- 《墨经》中记录了世界上最早的光学知识。包括影的定义和生成，光的直线传播性和针孔成像等。(圭表、日晷)
- 北宋著名学者沈括所著《梦溪笔谈》被认为是中国科技史的里程碑，里面不仅记录了多种光学现象，包括平面镜、凹球面镜和凸球面镜中物和像的关系。
- 欧几里德在其著作《光学》一书中提出触须学说：(1)正确反映了光的直线传播规律；(2)错误：人眼能发出光线。

**人物**：
- 墨子 (公元前468~376年)
- 欧几里德 (公元前330~275年)


### 经典光学 (16世纪-19世纪)
- 伽利略望远镜 (1609年)，开普勒望远镜 (1611年)
- 光的微粒学说 (牛顿)，折射定律 (1621年)，费马原理 (1657年)
- 惠更斯 (荷兰, 1629~1695)：光的波动理论的创始人，提出了“光是‘以太’中传播的波动”理论和次波假设 (1678年惠更斯原理)
- 托马斯·杨 (英, 1773~1829)：最先利用干涉原理解释了白光下的薄膜颜色，设计并完成了著名的杨氏双缝干涉实验，提出了光是横波的假设。
- 菲涅耳 (法, 1788~1827)：利用杨氏干涉原理补充惠更斯原理而提出了惠更斯-菲涅耳原理，解释了光的直线传播定律和衍射现象。建立了菲涅耳公式。(泊松亮斑)
- 马吕斯 (法, 1775~1812)：发现了光的偏振现象，建立了马吕斯定律。
- 麦克斯韦 (英, 1831~1879)，赫兹 (德1857~1894)：光是电磁波。


### 量子光学
发现经典电磁理论在研究光与物质的相互作用时的缺点，证实了以太的不存在，建立了光的量子理论，圆满解释了黑体辐射、光电效应和康普顿效应现象；提出了光的波粒二象性。

- 黑体辐射问题：普朗克 (德, 1858-1947)
- 解释光电效应：玻因斯坦 (美, 1879-1955)
- X射线散射实验：康普顿 (美, 1892-1962)
- 提出物质波假说：德布罗意 (法, 1892-1989)

- 频率 \( v \)
- 波长 \( \lambda = \frac{u}{v} \)
- 横波

## 1. 概论 - 几何光学
几何光学是光学中发展最早的一部分。它所研究的中心问题是物体经光学系统成像的问题。它是以光线概念为基础,研究光的传播规律和成像规律。

适用条件: 传播时遇到障碍物的尺寸远大于波长。

### 几何光学内容
- 光学系统成像性质
  - 像的亮度
  - 像的相似性
  - 像的大小
  - 像的色彩还原性
  - 像的信息
- 像差
- 简单仪器
- 成像规律
- 像质评价
- 传播规律
- 直观、简便、可靠
- 局限性、近似性

## 1. 概论 - 几何光学基本概念
### 光源分类
- 以外形抽象
- 点光源：几何上的点，只有空间位置无体积的光源
- 扩展光源
  - 面光源
  - 线光源

- 光线：表示光波能流传播方向的几何线
- 光束：有一定几何关系的一些光线的集合（平行、汇聚、发散）
- 波面：是电磁波位相相同点的集合（在各向同性媒质中，能量传播方向垂直于波面，即光线是波面的法线方向。）
- 光源：任何发光的物体：太阳，汞灯，日光灯等

### 光线与波面
- 平面波
- 波面
- 波线
- 柱面波
- 球面波

### 光束类型
- 同心光束：空间具有一定关系的光线的集合。
  - 会聚光束：沿着光的传播方向,光束的截面积不断减小
  - 发散光束：沿着光的传播方向,光束的截面积不断增大
  - 像散光束：各光线或光线的延长线不能相交于一点，但具有一定关系
- 非同心光束

## 2. 几何光学的基本定律
- **光直线传播定律**：光在各向同性、均匀介质中沿直线传播。 (影子, 小孔成像)。反之, 非均匀介质中, 光以曲线传播, 向折射率增大方向弯曲。
- **光的独立传播定律**：当多束非相干光线通过空间某点时，各光线的传播保持自己的特性不变，不受其他光线的影响。
- **折射定律、反射定律**：
  \[
  n \sin i = n' \sin i'
  \]
  \[
  n' = -n \quad i' = -i
  \]
- **光路可逆性原理**：光线在介质中沿某一路径传播，当光线反向时必沿同一路径逆向传播。

### 定律成立条件
- 均匀介质：折射率处处相等，不是位置的函数；
- 各项同性：折射率各向相等，不是方向的函数；
- 光强不强：满足线性叠加规律；
- 线度波长：线度大于所研究光的波长；

### 费马原理
费马原理是一个描述光线传播行为的原理。

- 光程定义：光线路径的几何长度与所经过的介质折射率的乘积。
  - 真空中： 光程 = l
  - 均匀介质： 光程 (QP) = n*l 沿L
  - 分段均匀： \( L(QP) = n_1 \Delta l_1 + n_2 \Delta l_2 + n_3 \Delta l_3 \)
  - 对于非均匀介质： \( L(QP) = \int_{Q}^{P} n(r) ds \) 沿L

Fermat于1679年提出的，从光程的观点描述光传播的规律:光传播的实际路径是使光程取极值(极小值、极大值或稳定值)。

\[
\delta L = \delta \int_{A}^{B} n dl = 0
\]

- 光的直线传播定律，反射定律，折射定律，光路可逆性原理等都可由费马原理证明。
- 费马原理规定了光线传播的唯一可行的路径。

### 费马原理示例
- 反射定律推导
- 折射定律推导
- 椭面反射镜
  \[
  F_1M + MF_2 = 2a \quad F_1F_2 = 2c
  \]
  \[
  \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 \quad a^2 - c^2 = b^2
  \]
- 半球面反射镜 (极大值)

### 费马原理与成像
- 严格等光程 ⇔ 严格成像
- 近似等光程 ⇔ 近似成像
- 非等光程 ⇔ 不成像
- 实像、虚像

## 3. 物像虚实
- 实物：光线进入人眼
- 实像：光线进入人眼
- 虚像：光线进入人眼
- 人眼在像点发光范围内可见它

### 虚物点的形成
- 入射光线延长线的交点

- 发出入射光线的是物，由出射光线形成的就是像。
- 光学成像系统
  - 实物 实像
  - 虚物 虚像
- 有光线真正的交点就是实。

- 虚物不能人为设定,只能由系统元件产生。
- 通常在光学设计中,系统至少要有一个实像,否则会抑制系统的功能。
- 物空间：物所在的空间。
- 像空间：像所在的空间。
- 物像空间是按照光线所在的空间来判断。
- 物像空间是可以无限延伸的。
- 共轭: 互成物像关系。

## 5. 单个球面折射公式
意义：最基本的成像元素, 其成像规律也是最简单、最基本的。

\[
n', r, L, U, n \rightarrow L', U'
\]

- 转面公式

- 当折射球面确定时, 出射光线 \( L' \), 写入射光线的位置、入射角有关。
- 一个物点经过单个折射球面成像后, 不会汇聚于一点。\( L'(L) \)
- 点物不能成点像。

### 完善像
像与物只有大小的变化，而没有形状的改变。

完善成像的条件：
- 物.像空间符合:点对应点
- 直线对应直线
- 平面对应平面

### 近轴光线
靠近光轴的光线，入射孔径角< 5° 的光线。

理想像：
- 物体以近轴光成像时，所成的像为理想像。

思考：仅与物距有关，所以可以由一个理想物点生成一个理想像点

（上图为类似几何示意图，显示球面折射的物像关系、角度和距离标签。）

#### 1. 起始关系（从图片几何和斯涅尔定律）

- **斯涅尔定律**：n sin I = n' sin I'（或图片中 sin I' = (n/n') sin I）。
- **几何关系（从正弦定律导出）**：
  - 在三角形 O P C 中应用正弦定律（law of sines），得到 sin I = (L - r)/r sin U。
  - 类似，在三角形 A P C 中，sin I' = (L' - r)/r sin U'。
- **角度关系**（图片提供，用于验证几何一致性）：
  - U' = I + U - I'（出射角与入射角的关系）。
  - φ = I' + U' - I + U（中心角 φ，近似为高度 h / r）。

#### 2. 代入斯涅尔定律

将几何关系代入斯涅尔定律：
$$
n \cdot \frac{L - r}{r} \sin U = n' \cdot \frac{L' - r}{r} \sin U'
$$
简化（取消 r）：
$$
n (L - r) \sin U = n' (L' - r) \sin U'
$$

#### 3. 近轴近似（小角度近似）

假设角度 U, U', I, I', φ 小（近轴光），则 sin x ≈ x（以弧度计）：
$$
n (L - r) U = n' (L' - r) U'
$$

#### 4. 引入轴向距离和高度 h

图片中的 L 和 L' 是光路长度 OP 和 PA，在近轴近似下，L ≈ l（轴向物距，正值表示物在左侧），L' ≈ l'（轴向像距，正值表示像在右侧）。忽略 h² 高阶项（h 为 P 点高度）。

同时，U ≈ h / l（物方孔径角），U' ≈ h / l'（像方孔径角）。代入：
$$
n (l - r) \cdot \frac{h}{l} = n' (l' - r) \cdot \frac{h}{l'}
$$
取消 h（假设 h ≠ 0）：
$$
n (l - r) / l = n' (l' - r) / l'
$$

#### 5. 方程整理

展开：
$$
n \left(1 - \frac{r}{l}\right) = n' \left(1 - \frac{r}{l'}\right)
$$
移项：
$$
n' \frac{r}{l'} - n \frac{r}{l} = n' - n
$$
除以 r：
$$
n' / l' - n / l = (n' - n) / r
$$
这即为单个球面折射公式（标准形式）。注意：在符号约定中，若物距 l 取负值（实物左侧），公式保持一致。

#### 6. 角度关系的验证（可选补充）

图片中的 U' = I + U - I' 和 φ = I' + U' - I + U 可用于验证一致性。在近轴下，I ≈ (l - r)/r U, I' ≈ (l' - r)/r U'，代入 U' = I + U - I' 可得与步骤 5 一致的关系，但不需直接用于主推导（它确保几何闭合）。

#### 7. 特殊情况

- 若 l → ∞（平行入射），则
  n' / l' = (n' - n) / r，l' = n' r / (n' - n)（像方焦距）。
- 若 n = 1（空气），公式简化为 n' / l' - 1 / l = (n' - 1) / r。

此推导基于近轴条件，适用于完善像计算。

### 阿贝不变量
\[
Q = n' \left( \frac{1}{r} - \frac{1}{l'} \right) = n \left( \frac{1}{r} - \frac{1}{l} \right)
\]

\[
\frac{1}{r}(n' - n) = \phi \quad \text{光焦度}
\]

若 \( l = -\infty \)，\(l'\)即为焦距

### 垂轴放大倍率
\[
\beta = \frac{y'}{y} = \frac{l' - r}{-l + r}
\]

符号的意义：

- (1) \(\beta = \frac{n'}{n} > 0\) 正像; 物像虚实相反; \(\beta < 0\) 倒像; 物像虚实一致。
- (2) \(|\beta|>1\) 放大; \(|\beta|<1\) 缩小;
- (3) \(\beta = \frac{n_l'}{n_l}\) 垂轴放大倍率随共轭面改变;
- 一对垂轴共轭面内的垂轴放大倍率相同。

垂直于光轴的同一平面上的各部分放大率相同。
### 轴向放大倍率
光轴上一对共轭点沿轴向移动量之间比例关系。

\[
\alpha = \frac{dl'}{dl} = -\beta^2 = \frac{n'}{n} \beta_1 \beta_2
\]

- (1) \(\alpha > 0\) 物像移动方向相同。
- (2) \(\alpha \neq \beta\)

### 角放大倍率
近轴区一对共轭光线像方孔径角与物方孔径角之比。

\[
\gamma = \frac{u'}{u} = \frac{l}{l'} = \frac{n}{n'} \cdot \frac{1}{\beta}
\]

三种放大倍率的关系： \(\alpha \cdot \gamma = -\beta\)

### 例题
已知一个光学系统的结构参数，\( r = 36.480 \) mm，\( n=1 \)，\( n'=1.5163 \)

已知一物体 \( y = 20 \) mm，\( l = -240 \) mm，\( l' = 151.83 \) mm，

现求 \(\beta\) 和 \( y' \) (横向放大率与像的大小)

解: \(\beta \approx -0.42\), \( y' = -8.4 \) mm

### 拉赫不变量
\[
nuy = n'u'y' = J
\]

- 像差与孔径(u)与视场(y)有关。像差越大, 系统结构越复杂, 造价越高。
- 拉赫不受量的重要性: 它数值的大小决定了光学系统设计的难易程度。


### 本章小结 - 基本概念阐释

以下是对PPT中列出的基本概念的详细阐释。这些概念是几何光学的基础，用于描述光学系统中的成像过程、光线传播和像质评价。我将逐一解释每个概念的定义、物理意义、相关公式（如果适用）以及在光学系统中的应用示例。解释基于近轴光学近似（paraxial approximation），即光线靠近光轴且入射角小（通常<5°），以确保理想成像。

#### 1. 物、像、虚、实
- **物（Object）**：指光学系统中发光或反射光的物体，是光线的起点。实物是实际存在的物体，光线从其表面发出进入光学系统；虚物则是由前一个光学元件形成的虚像，作为后续系统的输入（例如，在复合系统中，前透镜的虚像成为后透镜的虚物）。物可以是点物、线物或面物。
- **像（Image）**：光学系统对物进行成像后的结果，是光线汇聚或发散的点。像的位置、大小和性质取决于系统的参数（如折射率、曲率半径）。
- **虚（Virtual）与实（Real）**： 
  - **实物/实像**：光线实际交汇于一点，人眼能直接接收到光线（例如，投影仪屏幕上的图像是实像）。
  - **虚物/虚像**：光线或其延长线交汇于一点，但光线并不实际通过该点，需要通过延长线判断（例如，平面镜中的像为虚像）。虚像不能在屏幕上捕捉，但可以通过眼睛观察。
- **物理意义**：区分虚实有助于判断像的位置（实像在系统后方，虚像可能在系统前方）和成像类型（正像或倒像）。在设计中，通常至少需一个实像以实现实际功能，如摄影。
- **应用示例**：在凸透镜中，物体置于焦距外形成实像（倒立、缩小）；置于焦距内形成虚像（正立、放大）。

#### 2. 物空间、像空间
- **定义**：物空间（Object Space）是物体所在的空间，光线从这里进入系统；像空间（Image Space）是像形成的空间，光线从系统出射后在这里汇聚。两个空间可无限延伸，按光线传播方向划分。
- **物理意义**：物空间和像空间的折射率可能不同（n vs n'），影响光线的传播速度和弯曲。它们是相对的：在复合系统中，一个系统的像空间可成为下一个系统的物空间。
- **应用示例**：在空气-玻璃界面（n=1, n'=1.5），物空间为空气，光线进入玻璃后进入像空间。虚物可能出现在像空间的反向延伸中。

#### 3. 共轭
- **定义**：如果一个点作为物时，其对应像点固定，且反之亦然，则该对点互为共轭点（Conjugate Points）。扩展到面，即共轭面。
- **物理意义**：共轭关系确保点对点对应，是完善成像的基础。满足光路可逆性：光线从物点到像点，反向传播仍沿原路径。
- **应用示例**：在薄透镜中，物距l和像距l'满足1/l + 1/l' = 1/f（透镜公式），l和l'对应的点即共轭点。

#### 4. 完善像
- **定义**：也称理想像（Perfect Image），指像与物在形状、大小比例上完全相似，无畸变。条件：点对应点、直线对应直线、平面对应平面。
- **物理意义**：实际系统中，由于像差（aberrations），完善像仅在近轴区近似实现。非近轴光会导致像散、彗差等，破坏完善性。
- **应用示例**：单个球面无法形成完善像（出射光不汇聚于一点），需多个元件组合（如双胶合透镜）来校正。费马原理（光程极值）用于证明完善成像的等光程条件：严格等光程 → 严格成像；近似等光程 → 近似成像。

#### 5. 垂轴放大倍率、轴向放大倍率、角放大倍率
这些是描述像相对于物的尺寸和角度变化的量度，均在近轴区定义。
- **垂轴放大倍率（Transverse/Lateral Magnification, β）**：像高y'与物高y之比，β = y'/y = (n l') / (n' l)。符号表示像的正倒：β > 0为正像（虚实相反），β < 0为倒像（虚实相同）；|β| > 1为放大，|β| < 1为缩小。
  - **物理意义**：描述横向（垂直光轴）尺寸变化，随共轭面变化，但同一平面上各点放大率相同。
  - **示例**：在例题中，β ≈ -0.42，表示倒立缩小像，y' = -8.4 mm。
- **轴向放大倍率（Longitudinal/Axial Magnification, α）**：沿光轴方向，像的微小位移dl'与物的dl之比，α = dl'/dl = -β² = (n'/n) β₁ β₂。
  - **物理意义**：描述纵向深度变化，α ≠ β说明像在轴向和垂轴上不成比例（导致畸变）。α > 0表示物像移动方向相同。
  - **示例**：若β = -2，α = -4，表示轴向放大4倍（但方向相反）。
- **角放大倍率（Angular Magnification, γ）**：像方孔径角u'与物方u之比，γ = u'/u = l/l' = (n/n') (1/β)。
  - **物理意义**：描述光束角度变化，影响系统的视场和亮度。三种放大率关系：α · γ = -β。
  - **示例**：在显微镜中，高γ用于放大小角度细节。

#### 6. 拉赫不受量
- **定义**：也称Lagrange Invariant或Helmholtz-Lagrange Invariant，J = n u y = n' u' y'（常数）。
- **物理意义**：在整个光学系统中不变，表征系统的“通光能力”或étendue。J的大小决定像差校正难度：J越大，像差越大，系统设计越复杂、成本越高。结合孔径角u（aperture）和视场y（field），用于评估系统性能。
- **应用示例**：在设计中，J用于计算光通量或衍射极限。像差与u和y相关，J帮助平衡孔径（亮度）和视场（覆盖范围）。
